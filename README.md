# ZKON Mina zkApp

This repository contains the zkapps that powers the ZKON oracle ecosystem on Mina chain.

## How to integrate

Check [HOW-TO](HOW-TO.md)

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
